# Datapulsify - SEO Data Analytics Platform

A modern web application built with Vite, TypeScript, React, shadcn-ui, and Tailwind CSS for seamless Google Search Console data integration and analysis.

## Project Overview

Datapulsify is a comprehensive SEO data analytics platform that enables users to sync Google Search Console data directly to Google Sheets, providing powerful insights and automation tools for SEO professionals and marketers.

## Technologies Used

- **Frontend**: Vite + TypeScript + React
- **UI Components**: shadcn-ui
- **Styling**: Tailwind CSS
- **Authentication**: Supabase Auth
- **Database**: Supabase (PostgreSQL)
- **Animations**: Lottie React
- **Deployment**: Vercel

## Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Supabase account

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```

4. Configure your `.env.local` file with:
   ```
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm run preview
```
